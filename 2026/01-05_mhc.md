# mHC: Manifold-Constrained Hyper-Connections

* Authors: DeepSeek
* Link: https://www.arxiv.org/abs/2512.24880
* Date: 2025-12-31

## Notes

* Hyper-Connections (HC): extends the residual connection by expanding the residual stream width and diversifying
  connectivity patterns; performance gain, but training instability, restricted scalability, memory access overhead
* Manifold-Constrained Hyper-Connections (mHC): project the residual connection space of HC onto a specific manifold to
  restore the identity mapping property
